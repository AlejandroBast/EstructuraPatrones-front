# Frontend (Next.js)

Este repo contiene la parte frontend de la aplicación (Next.js).

Estructura mínima:
- `app/` páginas y layout
- `components/` componentes UI
- `public/` activos estáticos
- `styles/` estilos

Para desarrollo local, copia `.env.local.example` a `.env.local` y rellena las variables necesarias.
